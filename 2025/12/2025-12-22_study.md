🔵 **iframe（アイフレーム）とは？**

**➡ 一言でいうと：「Webページの中に、別のWebページを表示するための枠」**

HTML のタグの一つで、外部サイトのコンテンツ（YouTube、Google Map、Twitter など）を

**自分の Web ページの中に埋め込むための仕組み** です。

---

# 🖼 iframe のイメージ

```
┌───────────────────────┐
│   自分のWebページ       │
│   ┌──────────────────┐ │
│   │ iframe（外部ページ）│ ← 中身はYouTubeなど
│   └──────────────────┘ │
└───────────────────────┘

```

---

# 🔧 **YouTube が iframe を使う理由**

YouTube の動画はそのまま HTML に貼れません。

そこで YouTube は「iframe を使って動画を再生してね」という仕組みを提供します。

例：

```html
<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/dZ2dcC4OnQE"
  frameborder="0"
  allowfullscreen>
</iframe>

```

この iframe があることで：

- 画面の中にYouTubeのプレイヤーが表示される
- 再生・停止などは YouTube 側が処理してくれる
- 見た目は自分のページにあるように見える

という便利な仕組みになっています。

---

# 🧠 iframe のポイントまとめ

| 概念 | 説明 |
| --- | --- |
| ❇ iframe | Webページの中に別のページを表示する枠 |
| ❇ 外部コンテンツ埋め込み | YouTube、GoogleMap、Qiita、Twitter など |
| ❇ 利点 | 自分のサーバーで動画を持たずに済む・外部が提供する機能を使える |
| ❇ Rails での使い方 | View に iframe タグを埋め込むだけ |

---

# 🔍 iframe が使われる主な場面

- YouTube 動画を埋め込みたいとき
- Google Map をサイトに表示したいとき
- Note や Qiita を埋め込むとき
- 外部のフォーム（Google Form など）をサイトに置くとき

---

# 🎯 Twitter（X）は iframe を使わないの？

実は Twitter は **iframe ではなく script（widgets.js）で表示** します。

```html
<blockquote class="twitter-tweet">
  <a href="https://twitter.com/user/status/123"></a>
</blockquote>
<script async src="https://platform.twitter.com/widgets.js"></script>

```

ここが YouTube と違うポイント！
