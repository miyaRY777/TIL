### SQL

##### CURRENT_TIMESTAMP関数

説明：
現在の日時、（年、月、日、時、分、秒）を取得したいときに使う関数です！
また、日付（年、月、日）だけならCURRENT_DATE関数で、時刻（時、分、秒）だけならCURRENT_TIME関数！
つまり
今日の日付を取得したいなら、CURRENT_DATEっと記述すればいい！

基本形：
```sql
CURRENT_TIMESTAMP → 現在の日時（年、月、日、時、分、秒）
CURRENT_DATE関数　 → 現在の日付（年、月、日）
CURRENT_TIME関数　 → 現在の時刻（時、分、秒）
```

具体例：現在の日付を取得して登録する
![[スクリーンショット 2025-11-03 22.45.56.png]]

SQL：
```sql
INSERT INTO 家計簿
VALUES (CURRENT_DATE, '食費', 'ドーナツを買った', 0, 260)
```

出力：
|2025-11-03|食費|ドーナツを買った|0|260|が追加される！


##### CAST関数

説明：
あるデータの型を別のデータの型に変換したいときに便利なものがCAST関数！！！

基本形：
```sql
CAST(変換する値 AS 変換する型)　→ 変換後の値
```

具体例：
出金額 + '円'としたいとき、出金額は数値で、'円'は文字列なのでデータ型が一致しません！
なのでこれを解消するために
```sql
CAST(出金額 AS VARCHAR(20)) + '円'
```
