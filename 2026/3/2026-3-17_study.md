# 今日理解できたこと
## Webの本質

* Webは「URI・HTTP・HTML」で成り立っている
* URI = どこ（リソース）
* HTTP = どうやって（通信）
* HTML = 何（中身）

## Webの流れ

* URL入力 → リクエスト送信 → サーバー処理 → レスポンス → 表示 → 次のリンクへ

## Stimulusの理解

* HTMLにJSの振る舞いを“後付け”するフレームワーク
* controller = 振る舞いの単位
* action = イベントとメソッドの接続
* target = 操作対象の要素
* value = controllerに渡す設定値
* data属性でHTMLとJSを結びつけるのが特徴

## JavaScriptオブジェクト

* オブジェクト = 意味を持ったデータの集合
* 現実世界のモデリング（postなど）
* アクセス方法

  * ドット記法 → シンプル
  * ブラケット記法 → 動的
* constの本質

  * 参照（場所）を固定しているだけ
  * 中身は変更可能
* 実務では「配列 × オブジェクト」が基本
