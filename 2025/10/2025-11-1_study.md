## 学習内容

### SQL

##### TRUNC関数

説明：
四捨五入ではなく、指定桁で切り捨てたいときに利用する関数です！

基本形：
```ruby
TRUNC(数値を表す列, 有効とする桁数)
```

具体例：100円単位の出金額を切り捨てで取得する！

![[スクリーンショット 2025-11-01 22.59.14.png]]

SQL：
```sql
SELECT 出金額, TRUNC(出金額, -2) AS 百円単位の出金額
  FROM 家計簿
```

出力：
```sql
出金額|100年単位の出金額|

|380|300|
|0|0|
|2800|2800|
|5000|5000|
|7560|7500|
  ```


##### POWER関数

説明：
ある値を２乗、３乗と計算したいときに、用いる便利な関数のことです！

基本形：
```sql
POWER(数値を表す列, 何乗するかを指定する数値)
→数値を指定した回数だけ乗じた結果
```

具体例：
POWER(出金額, 3)と記述すると、（出金額 * 出金額 * 出金額）となり、出金額を３乗した値が出力される
