# `typeof`（タイプオブ）とは？

## 結論（PREP）

`typeof`（タイプオブ）とは、**値の「型」を文字列で返すJavaScriptの演算子**です。  
デバッグや入力チェックで「いま何が入っているか」を素早く確認できます。
## 理由

JavaScriptは同じ変数に、数や文字など別の種類の値が入りえます。  
そのため「想定どおりの型か」を確認できると、バグを減らせます。
ただし `typeof null` が `"object"` になるなど、**クセ（落とし穴）**もあります。
## 具体例

```js
typeof 10        // "number"
typeof "10"      // "string"
typeof undefined // "undefined"
typeof null      // "object"  ← 注意（null判定は === null を使う）
```

`null` を判定したいときは、`typeof` ではなく **`value === null`** が安全です。

## 結論（言い換え）

`typeof` は、**「値の種類を文字列で確認するための道具」**です。
ただし `null` は例外なので、**`=== null` で判定する**のが基本です。
