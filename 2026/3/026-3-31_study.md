# 今日やったこと

# 今日理解できたこと

* Active Record Callbacks は「保存の前後に自動処理を差し込む仕組み」
* before_create を使うと、レコード作成前に値を補完できる
* CurrentAttributes を使うと、ログインユーザーなどをどこからでも参照できる
* コントローラでセット → モデルで利用、という流れになる
