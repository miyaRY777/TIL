## 学習内容

### SQL

##### COALESCE関数

説明：
まず、複数の引数を受け取り、それらを左から順番にチェックします
その中から最初に見つかった「NULL」ではない引数を返すっといった動作をする関数です！
また、この関数を使うタイミングは
NULLであることを明示的に表示したいからです！

基本形：
```sql
COALESCE(列や式１, 列や式２, 列や式３, ・・・)

/*任意の数の引数を指定できますが、すべての引数の方を一致させる必要があります！*/
```

具体例：

![[スクリーンショット 2025-11-04 22.01.46.png]]

SQL：
```sql
SELECT 日付, 費目,
       COALESCE (メモ, '(メモはNULLです)') AS メモ,
       入金額, 出金額
  FROM 家計簿
```

出力：
```sql
|日付|費目|メモ|入金額|出金額|
|---|---|---|---|---|
|2024-02-03|食費|自分へのご褒美|0|380|
|2024-02-11|教養娯楽費|(メモはNULLです)|0|2800|
|2024-02-14|交際費|(メモはNULLです)|0|5000|
```

##### 集計関数とは

説明：
検索結果のデータを集計するための関数です！

今までの関数との違い：
LENGTHやCOALESCEと言った関数は、各行に対して1行ずつ順番に処理をします！
一方、集計関数は
集計の対象のすべての行に対して、まとめて1回の処理をし、その結果を1行にまとめます！


##### SUM,MAX,MIN,AVG 関数

説明：
```sql
/*検索結果を集計する関数*/
 SUM(列 or 列名を含む式)　→ 合計
 
 /*検索結果の最大値を返す関数*/
 MAX(列 or 列名を含む式)　→ 最大値
 
 /*検索結果の最小値を返す関数*/
 MIN(列 or 列名を含む式)　→ 最小値
 
 /*検索結果の平均値を返す関数*/
 AVG(列 or 列名を含む式)　→ 平均値

```

具体例：
![[スクリーンショット 2025-11-04 22.36.19.png]]

SQL：
```sql
SELECT SUM(出金額) AS 合計出金額,
       AVG(出金額) AS 平均出金額,
       MAX(出金額) AS 最も大きな散財,
       MIN(出金額) AS 最も少額の支払い
  FROM 家計簿
 WHERE 出金額 > 0
```

出力：
```sql
|合計出金額|平均出金額|最も大きな散財|最も少額の支払い|

|15740|3935.0000000000000000|7560|380|
```
