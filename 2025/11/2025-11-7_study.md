## 学習内容

### SQL

##### 費目ごとに集計結果を得る方法

説明：
費目ごとに集計結果を得たい場合、2つのアプローチがあります！
1つ目は、SELECT文で、「食費」や「給料」などの個々の費目に対して、WHERE句で行を絞り込む方法がありますが、完全に力技で非常に手間です笑
なので、2つ目の方法であるGROUP BY句を利用して、絞り込む方が簡潔です！

これは
SQLのグループ化と言う機能1つで、指定した基準でいくつかのまとまりに分けることができ
グループごとに集計を行います！

基本形：グループ化して集計する基本構文
```sql
SELECT グループ化したい基準の列名, 集計関数
FROM　テーブル名
WHERE　絞り込む条件(任意)
GROUP BY　グループ化したい基準の列名
```


具体例：費目でグループ化してそれぞれの合計を求める

家計簿
![[スクリーンショット 2025-11-07 23.02.44.png]]

SQL：
```sql
SELECT 費目, SUM(出金額) AS 費目別の出金額合計
  FROM 家計簿
 GROUP BY 費目

```

出力：
```sql
|費目|費目別の出金額合計|

|食費|380|
|給料|0|
|交際費|5000|
|教養娯楽費|2800|
|水道光熱費|7560|
```


##### グループ化集計の流れ

1. 検索
   今まで通り、WHERE句によって絞り込み、検索結果を出力します！
2. グループ化
   基準を決め、検索結果からグループごとに分類します！
3. 集計、列選択
   各グループを集計し、SELECT文で列を絞り込みます！
4. 集計結果
   集計関数の結果表の行数は、必ずグループの数と一致する
